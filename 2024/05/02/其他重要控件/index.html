Qt其他重要控件

QWidget是其他组件的容器，也即父类

QCheckBox-复选框

  1. QCheckbox("text")定义一个复选框

  2. 可以通过setChecked(True/False)和setChecked(Qt.Checked...)来设置复选框的选中状态

  3. stateChanged信号可以connect函数，状态发生变化触发函数

  4. checkbox.setTristate(True/False)可以定义是否含有半复选状态

QRadioButton-单选

  1. QRadioButton("text")
  2. A radio button emits the toggled() signal when it is switched on or off. toggled()信号在该单选选项状态变化时触发
  3. 相同父类的单选属于同一组,同一组中只会有一个被选中
  4. isChecked()检测按钮的开关

QComboBox-下拉列表

  1. QComboBox(self/parent)
  2. 在列表中添加item，addItem(相当于append)和InsertItem(index,item)
  3. 取当前选中的Item的currentData(),currentIndex(),currentText()
  4. 选项改变时产生QcomboBox.activated信号

QSpinBox-计数器

属性：

  1. QspinBox("参数可以设置属性如minimum=1")
  2. spinbox中的数值发生变化后会释放valueChanged信号,该信号连接的函数会接受到一个返回值，为当前spinbox中的数字，还有textChanged信号，返回Qstring

QDateTimeEdit/QDateEdit/QTimeEdit

属性：

  1. QDateTimeEdit(self,calendarPopup=True/False) ,后面的参数若为true则展示日历
  2. 时间编辑完成时释放editingFinished信号，值变化释放dateTimeChanged信号
  3. QdateTime格式转日期格式value.toString("yyyy-MM-dd HH:mm")
  4. dateTime ()方法取得当前的组件中的value

其他两种类似QDateTime

转格式不同：date为str(value.toPyDate())，取为date ()

time 为str(value.toPyTime())，取为time()
