Qt多线程和多进程

Qt applications are event-based. When you call the exec() method, it starts an event loop and creates a thread that is referred to as the main thread.

多线程

  1. 进程就是运行着的程序,每个进程里面至少包含一个 线程 。线程是操作系统创建的，每个线程对应一个代码执行的数据结构，保存了代码执行过程中的重要的状态信息。进程开始时，OS就自动的创建一个主线程

  2. 多核cpu，让多个核同时执行同一程序，加快执行速度

  3. 添加新进程：

  4. join函数：线程a调用了b的join函数，线程A就会停止继续执行代码，等待线程B结束。 线程B结束后，线程A才继续执行后续的代码。一般A为主线程

  5. 有时多个线程同时进行某个数据对象，可能会导致冲突产生覆盖了其他线程的操作，因此引入线程锁lock(),在线程入口函数中先获取线程锁lock.acquire(), 别的线程已经申请到了这个锁， 并且还没有释放，那么 线该线程的代码就在此处等待拥有线程锁的线程释放锁，不去执行后面的代码。线程操作完数据时释放线程锁lock.release()

  6. 主线程和子线程 都结束，程序才会结束。
